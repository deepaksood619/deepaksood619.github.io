## Sets

Application - Exception filter

- Read in a list of words from one file
- Print out all words from standard input that are `{in, not in}` the list

    - Blacklist a set of strings or objects

- Spell checker
- Browser
- Parental controls
- Chess
- Spam filter
- Credit cards
