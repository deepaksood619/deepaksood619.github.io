**For discovering startup opportunities:**

2. Review vetted ideas for MVP-ready concepts
3. Browse brainstorm lists for unexplored domains

1. Check research frameworks for templates

3. Review competitor analyses

1. Read guidance section
2. Study founders journey
3. Review business creation fundamentals

This is an LLM-maintained wiki. When adding new research:

- Update research-index.md
