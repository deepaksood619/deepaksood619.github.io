## AI/ChatGPT Ideas

- **CodeRabbit Alternative** - Better pricing model (pay as you go instead of per-seat)
- **SMS GPT Services** - Via SMS, get GPT assistance in Hindi
  - MedGPT, FinanceGPT, LearningGPT, CounsellorGPT, ProfessorGPT
- **AI Interview Preparation Platform** - Live tutor/AI assistant for interview prep
  - Check communication skills, depth of knowledge
  - New questions every time during interviews/tests
  - Competitors: [Round1 AI Interview](https://www.grapevine.in/round1/interviews), [Remasto](https://remasto.com/), [Interviews by AI](https://interviewsby.ai/), [CareerElite](https://careerelite.ai/), [UnchartedCareer](https://unchartedcareer.com/)
- **AI Advertisement Generator** - Create adverts from text input
- **AI Data Analytics Platform** - Give schema and model to LLM, ask questions in plain English
  - Automate data analytics and data engineering tasks
  - Automatically write efficient SQL queries
  - No need for PowerBI reports / manual SQL / visualization creation
  - Auto-choose best graphs/plots/visualization, add refresh intervals

### Generative AI Content

- **AI Webtoons/Manga** - Webtoons / Blocktoons / MangaPlus but by AI artists
- **Subjective Paper Scoring** - Use ChatGPT for grading essays and assignments
- **AI Code Repository Assistant** - Full repository context, suggests file and line changes
- **AI Interview Captions** - During interviews, automatic captions sent to ChatGPT for answer suggestions
- **Hindi Book Summaries** - Audio content for Indian consumers
  - Book summaries with Hindi/English content
  - Multiple languages
  - Only 5-10 key points per book
  - Generate 100,000 book summaries
  - Could use Astro instead of Docusaurus

### AI Hiring & Recruitment

- **AI Hiring Portal**
  - Candidate screening using AI
  - Coding test platform end-to-end, free or cost competitive
  - Similar to Turing
  - **Hiring Agent:**
    - Sift through LinkedIn, Naukri, etc.
    - Call to get expectations
    - Set up interviews
- **One-time Payment ChatGPT** - API sharing agreement model

### AI Manager

- **Single-person management** using tech to manage large teams
- **AI Scrum Master**
  - Run daily scrum meetings
  - Ask: what you did yesterday, today, any blockers
  - Voice to text and vice versa
  - Ensure employees are progressing
  - Update tasks automatically
  - Reminders for no-shows
  - Ask team to be on-time
  - System Design, Architecture assistance
  - Documentation generation
  - Vision alignment

### All-in-One AI Platform

**One tool where you can do everything:**

- Chat, video call, tasks, documentation, sharing, email, status
- Follow-ups automated
- No manager needed for reviews (auto follow-ups, auto recognitions, auto reviews, auto hiring PIPs)
- For all teams powered by AI and LLM

**LLM-powered roles:**

- LLM HR and LLM manager
- LLM payroll, LLM account manager
- LLM sales, LLM marketing, LLM operations
- LLM customer support, LLM counselor, LLM companion

**Advanced AI features:**

- LLM that can finish your sentences
- Can read your mind
- Real-time conversation

### AI Voice & Conversational AI

- **Voice to AI chatbot** for various use cases
- **Conversational AI of important figures** (e.g., Mahatma Gandhi)
- **AI DJ with ML** - Learns from people's excitement with voice and image processing
  - Uses crowd sourcing to teach about different tastes
  - Differentiates between nations and preferences
  - Plays according to people present (romantic songs for couples, party songs for friends)
