## Core SaaS Principles

**Create best software companies with Indian labor** - Since labor is cheap in India (like Zoho)

**Don't pay for bits, pay for atoms** - Software is free/open source, pay for physical infrastructure and usage

### Problem with Current SaaS Pricing

- Per-user based (inefficient)
- Should be usage-based instead
  - Pay for 1 GB storage + 1 CPU + 2 GB RAM exactly
  - Plus some revenue margin

**Developer tools - pure software play**
