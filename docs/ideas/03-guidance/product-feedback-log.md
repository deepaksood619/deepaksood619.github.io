**Note:** This is a product feedback log, not startup research. Ideas here are improvements to existing products rather than new business opportunities.

## Android

### Notifications

- **Time-based notification control** - Enable notifications for an app for certain times/days
  - Example: Between working hours for Gmail work account
- **Notification preference time by users**
- **Pin notifications** - Keep important notifications at top

### Permissions

- **Internet access permission** - For outgoing data
  - What data is sent from apps to outside

### Copy/Paste

**New cut, copy, paste operation:**

- Paste copied item above selected text
- Cut the selected text simultaneously
- Helpful when two items must be swapped

Example use case:

```
Before:
- First
- Second

After swap operation:
- Second
- First
```

## YouTube

### Missing Features

- **No notes on videos** - Need annotation capability
- **Videos should be like books:**
  - Annotate and add notes
  - Search within videos
  - Arrange and organize
  - All features you can do with books

## Google Search

### Problems

- **Not real-time for technologies**
  - Newer technologies have low traffic
  - Older solutions with large user base rank higher
  - Should be opposite - prioritize latest solutions
  - **Cannibalization problem**

- **Gives links instead of solutions**
  - User has to choose from many links
  - Need direct answers

## Social Media

- **Fake News Dissemination Blocker**
