# Slides

## Boto3

- AWS SDK for Python developers
  - Software development kit
  - Interact with AWS easily through Python
- Supports Python 2 & 3
- Data-driven from the group up
- Up-to-date service API support
- Consistent interface
- Modern object-oriented API
